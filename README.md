# stalker

A discord bot that allows you to track players activity on a server using the [Battle Metrics](https://www.battlemetrics.com/) API


## Setup
Change the values in bot.properties to allow the bot to successfully run.
An example for Rustafied.com - AU Long
```
botToken=YOUR_BOT_TOKEN
serverId=1720719
```

## Usage

`/track [name]` - allows you to track that player. sends notifications inside the channel you ran the command in

`/players` - get the list of players on that server.
